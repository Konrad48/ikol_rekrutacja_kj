# Zadanie rekrutacyjne IKOL Konrad Janecki

## Frontend
Do frontendu użylem Nuxt.js, ponieważ mam już opracowany szybki boilerplate do tworzenia nowych projektów. Do stylizacji zastosowałem Tailwind CSS. Formularz ma już wypełnione pola i sprawdza poprawność wprowadzonych współrzędnych nie pozwalając na przesłanie niepoprawnych danych.

## Backend
W Laravelu stworzyłem jeden endpoint i konroler. W kontrolerze zawarta jest również walidacja danych wejściowych. Ponieważ to jest zadanie rekrutacyjne, nie dodałem żadnych dodatkowych funkcjonalności, takich jak np. logowanie, rejestracja czy autoryzacja, które normalnie bym obsłużył z Sanctum.
